'spawners.csv' currently contains estimates taken directly from original Cox-Rogers spreadsheet that was compiled for 2015 recovery plan.



* format of the sheet has been adjusted to include more information.
* Above Lonesome: is the SUM of all spawners and holding fish estimated above Lonesome Lake (live and dead). Some adjustments to these numbers have been made during escapement review.
* Below Stillwater: is the SUM of all spawners, holding and dead sockeye estimated to spawn between Stillwater Lake and Lonesome Lake, as well as observations of sockeye in Stillwater Lake. No river spawning sockeye below the lake are captured in this dataset.
* both estimates and the total estimate have been broken out by ratios if only total expanded estimates are provided. E.g. counted fish/total counted fish \* total estimate.
* Survey mode describes how data was collected.
* estimate method describes how the estimate was created
* Residency describes the residency time if an AUC was applied
* Reliability describes the confidence in the total estimate (if it was provided)
* Flag/follow-up notes any estimate that has suspicious entries, or requires revisiting
* the original notes from Matt Mortimer and Steve Cox-Rogers have been retained. Data source column was added by KM to note anything that is different about the source in a review.
* NuSEDs SIL and SEN status notes where /if in NuSEDs. 
* other KM comments: details on flags and follow- ups amongst others.



See River\_spawners for river spawners surveyed below lake system.
